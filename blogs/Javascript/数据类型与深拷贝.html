【Javascript基础系列一】 数据类型与深拷贝

Xiaoliuxxxx 2020-08-02 javascript

JS 中的数据类型以及深浅拷贝，在面试的时候经常被问及，这种基础的东西应该被前端开发者掌握。下面是我对数据类型的一些总结。

# 一.变量类型

# 1.1 基本数据类型有哪些？

number,string,boolean,undefined,null,symbol,bigint(最近新增)

# 1.2 基本数据和引用类型的区别？

基本数据保存在栈内存中，是按值访问的，引用数据类型保存在堆内存中，栈内存中保存了指针的引用，因此引用类型的值是按引用来访问的。

# 1.3 数据类型的检测有哪些方法

  • 1.typeof：基本数据类型中除了 null 均可以准确检测，还可以检测 function
typeof "str"; // "string"
typeof false; // "boolean"
typeof 1; // "number"
typeof null; // "object"
typeof undefined; //"undefined"
typeof Symbol(1); //"symbol"
typeof 1n; //"bigint"
typeof {}; // "object"
typeof []; // "object"
typeof (() => {}); //"function"

// 手写instanceof
function instance_of(L, R) {
  let R_P = R.prototype;
  let L_p = L.__proto__;
  while (true) {
    if (L_p === null) {
      return false;
    }
    if (L_p === R_P) {
      return true;
    }
    L_p = L_p.__proto__;
  }
}

  • 3.Object.prototype.toString.call() 这个检测方法是能够完全准确的检测到数据类型，且大部分库都有使用（安全可靠） 原理：

    Object.prototype.toString 方法的规范如下:在 toString 方法被调用时,会执行下面的操作步骤:

  1. 获取 this 对象的[[Class]]属性的值.
  1. 计算出三个字符串"[object ", 第一步的操作结果 Result(1), 以及 "]"连接后的新字符串.
  1. 返回第二步的操作结果 Result(2).
var x = {};
x.toString(); //"[object Object]"

# 三、深拷贝

# 3.1 什么是深拷贝，什么是浅拷贝？

浅拷贝：只拷贝对象的第一层 key-value 深拷贝：深层次拷贝对象

# 3.2 浅拷贝的实现方式

  • 1.Array.concat(),Array.slice()
  • 2.解构复制
  • 3.Object.assign()

# 3.3 深拷贝的实现

  • 1.JSON.parse(JSON.stringify()) 满足基本需求 该方法能够实现深拷贝，但是 function symbol 等数据类型会丢失

  • 2.递归实现深拷贝（常用）

function isObject(target) {
  let s = Object.prototype.toString.call(target);
  const type = s.match(/\[object (.*?)\]/)[1].toLowerCase();
  return type === "object";
}
function deepClone(target) {
  if (!isObject(target)) {
    return target;
  }
  var obj = {};
  for (let key in target) {
    if (isObject(target[key])) {
      obj[key] = deepClone(target[key]);
    } else {
      obj[key] = target[key];
    }
  }
  return obj;
}

解决这个问题的方案就是用一个全局的 map 去保存已经拷贝过的对象，当再次拷贝的时候直接返回就行不用再次递归拷贝。

function isObject(target) {
  let s = Object.prototype.toString.call(target);
  const type = s.match(/\[object (.*?)\]/)[1].toLowerCase();
  return type === "object";
}
function deepClone(target, map = new WeakMap()) {
  if (!isObject(target)) {
    return target;
  }
  if (map.get(target)) {
    return map.get(target);
  }
  var obj = {};
  map.set(target, obj);
  for (let key in target) {
    if (isObject(target[key])) {
      obj[key] = deepClone(target[key], map);
    } else {
      obj[key] = target[key];
    }
  }
  return obj;
}
